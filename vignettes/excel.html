The problem

Multiple header rows:

  • jurisdiction in row 2
  • office in row 3
  • candidate in row 4

The problem

Multiple-column or “merged” header cells:

  • Sheriff
  • Attorney
  • Treasurer

The ordinary approach

# required packages
library(medslcleaner)  # required
library(stringr)  # required
library(readxl)  # required
library(dplyr) 

# used for presentation in this vignette

Alternative solution

  1. Identify which cells are data and which are headers
  2. Define the relationships between data cells and header cells

Schema

Field schema define our expectations about data:

- name: votes
  title: Vote Count
  description: Number of votes received.
  source: Precinct returns for `jurisdiction`.
  type: integer
  constraints:
    required: true
